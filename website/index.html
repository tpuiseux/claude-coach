Quick Start Guide

Get up and running in minutes.

Getting Started

Claude Coach works with both Claude.ai and Claude Code. Choose your platform below.

  1. Download the skill
  2. Open Claude.ai Settings and enable "Code execution and file creation"
  3. Add Strava access: In the allowed domains list, add *.strava.com
  4. Install the skill: Scroll to "Skills" and click "Add skill", then upload coach-skill.zip
  5. Start chatting and ask Claude to create a training plan for your target event

  2. Open Claude Code in your terminal
  3. Run /install-skill and provide the path to the downloaded coach-skill.zip
